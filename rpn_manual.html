RPN 计算器使用说明书

本程序为命令行逆波兰（RPN）计算器，支持整数、浮点数、四则运算及扩展数学函数。

所有输入按「空格」分隔；一行即一次求值，回车得结果。

1. 安装与编译

echo """
  git clone https://github.com/yourname/rpn_calc.git
  cd rpn_calc
  g++ -std=c++17 rpn_calc.cpp rpn_engine.cpp -o rpn_calc
  """
  git clone https://github.com/yourname/rpn_calc.git
  cd rpn_calc
  g++ -std=c++17 rpn_calc.cpp rpn_engine.cpp -o rpn_calc

2. 启动

echo """
  ./rpn_calc
  === C++ RPN 计算器 ===
  支持操作: +  -  *  /  pow  sqrt  fib
  输入表达式（如 3 4 +），或 q 退出
  >
  """
  ./rpn_calc
  === C++ RPN 计算器 ===
  支持操作: +  -  *  /  pow  sqrt  fib
  输入表达式（如 3 4 +），或 q 退出
  >

3. 基础示例

加法

echo "> 3 4 +\n结果: 7"
> 3 4 +
结果: 7

乘法结合

echo "> 2 3 4 * +\n结果: 14"
> 2 3 4 * +
结果: 14

浮点除法

echo "> 22 7 /\n结果: 3.14286"
> 22 7 /
结果: 3.14286

4. 扩展运算

幂运算

echo "> 2 3 pow\n结果: 8"
> 2 3 pow
结果: 8

平方根

echo "> 2 sqrt\n结果: 1.41421"
> 2 sqrt
结果: 1.41421

斐波那契（第 n 项）

echo "> 10 fib\n结果: 55"
> 10 fib
结果: 55

5. 栈操作与调试

  • 连续运算自动复用栈内剩余数据
  • 输入非法字符或栈空时会提示错误，但程序不崩溃
  • 随时按 q 或 quit 退出

6. 错误提示对照表

| 错误信息 | 含义 |

|---|---|

| stack underflow | 运算符需要的操作数不足 |

| divide by zero | 除零 |

| sqrt of negative | 对负数开方 |

| fib index must >= 0 | 斐波那契下标为负 |

7. 批量模式（重定向）

支持管道一次计算多行：

echo "echo -e \"3 4 +\\n2 pow\" | ./rpn_calc"
echo -e "3 4 +\n2 pow" | ./rpn_calc

8. 常见问题

  • Q：必须空格分隔？ A：是的，Tab/多个空格也可。
  • Q：支持负数？ A：支持，如 -5 2 + 得 -3。
  • Q：括号怎么打？ A：RPN 无需括号，靠顺序即隐含优先级。

9. 源码与更新

开源地址：https://github.com/yourname/rpn_calc
